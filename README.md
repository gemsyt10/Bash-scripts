# Bash Scripts

Колекція моїх Bash-скриптів, які спрощують роботу в терміналі та автоматизують повсякденні задачі.

📦 Скрипти

🚀 Command Initer

Command Initer дозволяє швидко перетворити Bash-скрипт (".sh") на повноцінну команду терміналу.

Можливості:

- Ініціалізація Bash-скрипта як команди.
- Автоматичне надання прав на виконання.
- Можливість автоматично видалити інсталятор після встановлення.
- Підтримка Termux.

---

🔄 Auto Restarter

Auto Restarter автоматично перезапускає Node.js-застосунок після змін у файлі.

Можливості:

- Відстежує зміни у файлі.
- Автоматично перезапускає застосунок.
- Легка альтернатива "nodemon".
- Просте налаштування через "autore-conf.txt".

---

🖥️ Start Server ("ss")

Невелика утиліта для швидкого запуску ваших Node.js-проєктів.

Можливості:

- Збереження кількох серверів у списку.
- Запуск потрібного сервера однією командою.
- Перегляд списку доступних серверів.
- Вбудована довідка та інформація про версію.

---

Встановлення

Клонуйте репозиторій:

git clone <посилання-на-репозиторій>

Після цього ви можете запускати скрипти напряму або встановити їх як команди за допомогою Command Initer.

Вимоги

- Bash
- Node.js (для "Auto Restarter" та "ss")
- Termux (рекомендовано)

Ліцензія

Ви можете вільно використовувати, змінювати та вдосконалювати ці скрипти.
